# toDo-app-django-rest-framework-drf

This is a To-Do Application developed with Django Rest Framework, and mysql database is used.
<br>

<h2 style="color:red">End Points</h2> <br>
GET /api/tasks/ <br>
GET /api/tasks/{pk} <br>
POST /api/tasks/ <br>
PUT /api/tasks/{pk} <br>
DELETE /api/tasks/{pk} <br>
